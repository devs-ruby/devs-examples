# DEVS Examples

A library of common examples to go with Ruby DEVS implementation

## Installation

Install it as:

    $ gem install devs
    $ gem install devs-models
    $ gem install devs-examples

Inside of your Ruby program do:

    require 'devs'
    require 'devs/models'
    require 'devs/examples'

...to pull it in as a dependency.
